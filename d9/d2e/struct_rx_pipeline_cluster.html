Detailed Description

A structure describing a pipeline cluster; that is, an RxClusterDefinition in the context of a specific RxPipeline (in the same way as an RxPipelineNode is an RxNodeDefinition in the context of a specific RxPipeline). The RxCluster is the structure representing this and the RxClusterDefinition within RxPacket's at pipeline-execution-time
