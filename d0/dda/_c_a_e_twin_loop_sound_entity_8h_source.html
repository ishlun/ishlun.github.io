9 /* This class is used to control 2 sounds as one. There are 2 sounds

13  The time m_dwTimeToSwapSounds is calcualated as:
14  CAEAudioUtility::GetRandomNumberInRange(m_wPlayTimeMin, m_wPlayTimeMax);
15  Notice these sounds must be situated in same bank slot.
16  m_pBaseAudio is a pointer to audio which created twin sound. For example,
17  this could be CAEVehicleAudioEntity for playing skid sounds. */
